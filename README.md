# 🧠 Cognitive-Aware AI Chatbot  
**Prompt Engineering • LLM Evaluation • Cognitive Analysis**

An intelligent AI chatbot that goes beyond traditional responses by analyzing user emotion, confidence, and hesitation patterns to generate structured, adaptive outputs.

This project focuses on **prompt engineering, evaluation, and output control**, simulating real-world LLM system design.

---

## 🚀 Features

### 🧠 Cognitive Analysis
- Detects **emotion, confidence level, and hesitation patterns**
- Adapts responses based on user mental state

### 🔄 Prompt Engineering
- Structured prompt design (**Role, Context, Rules, Output Format**)
- Prompt versioning (**v1, v2, v3**) for experimentation

### 🧪 Prompt Evaluation Framework
Evaluates responses based on:
- Structure adherence  
- Relevance  
- Output consistency  

➡️ Helps reduce manual QA effort

### 🛡️ Guardrails & Reliability
- Fallback mechanism for weak model outputs
- Ensures consistent structured responses:
  - Advice  
  - Reasoning  
  - Question  

### 📊 Behavioral Tracking
- Tracks confidence over time  
- Visualizes user trends dynamically  

### 🎨 Interactive UI
- Built with **Streamlit**
- Clean, modern interface with real-time feedback  

---

## 🧠 Key Idea

This project is not just a chatbot.

---

## ⚙️ How It Works

1. User inputs a message  
2. System performs cognitive analysis:
   - Emotion detection  
   - Confidence scoring  
   - Hesitation detection  
3. A structured prompt is generated  
4. Model produces a response  
5. System evaluates output quality  
6. Fallback logic ensures structured response if needed
